# Ilustraciones SVG para la Boda

Esta carpeta contiene las ilustraciones SVG de alta calidad para la landing page.

## Estructura recomendada

Coloca tus archivos SVG aquí con estos nombres:

- `table.svg` - Ilustración para la sección "Celebración" (banquete/mesa)
- `location.svg` - Ilustración para la sección "Lugar" (finca/ubicación)
- `door.svg` - Ilustración para la sección "Alojamiento" (hotel)
- `ring.svg` - Ilustración decorativa (anillos)

## Fuentes recomendadas para SVG gratuitos

### Sitios especializados en bodas:
1. **LinvitInstant** - https://linvitinstant.com/es/
   - Escudos de boda elegantes en SVG
   - Logotipos florales minimalistas
   - Monogramas editables

2. **Freepik** - https://www.freepik.com/
   - Buscar: "wedding illustration svg"
   - Miles de ilustraciones gratuitas (requiere atribución)

3. **Flaticon** - https://www.flaticon.com/
   - Iconos y ilustraciones en SVG
   - Categoría: Wedding

4. **Undraw** - https://undraw.co/
   - Ilustraciones modernas y minimalistas
   - Personalizables con colores

5. **Open Peeps** - https://www.openpeeps.com/
   - Ilustraciones de personas (si necesitas)

## Estilo recomendado

- **Minimalista y elegante**
- **Líneas finas** (stroke width: 1-2px)
- **Colores**: Terracotta (#B87333), Taupe (#9C8E7C), Charcoal (#2D2D2D)
- **Temática**: Flores, hojas, elementos decorativos, mesas, puertas, anillos

## Personalización

Si descargas SVG editables, puedes:
- Cambiar colores para que coincidan con la paleta
- Añadir iniciales "D & P"
- Ajustar el tamaño según necesidad

## Fallback

Si un SVG no está disponible, el código automáticamente usará la versión AVIF como respaldo.
